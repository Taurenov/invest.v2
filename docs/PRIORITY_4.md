# Приоритет 4 — выполнено

## Что реализовано

- **Офлайн-режим транзакций**
  - локальный кэш списка транзакций;
  - офлайн-очередь операций `create/delete`;
  - авто-синхронизация очереди при восстановлении сети.

- **Импорт CSV**
  - кнопка `Импорт CSV` в экране транзакций;
  - формат CSV: `date,kind,amount,description`;
  - строки импортируются в API, при офлайне — уходят в очередь.

- **Backup / Restore**
  - экспорт локального backup в JSON (сессия, кэш, очередь, настройки);
  - импорт backup из JSON через экран настроек.

- **База под автообновления**
  - подключён `@tauri-apps/plugin-updater` (frontend);
  - подключён `tauri-plugin-updater` (Rust);
  - добавлена кнопка `Проверить обновления` в настройках.

## Проверка сборки

```powershell
cd apps\desktop
npm install
npm run build

cd src-tauri
cargo build
```

Обе сборки проходят.
